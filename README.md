# DFLR-IFC
Quantifies Data Fidelity Loss Ratio (DFLR) when exporting Revit models to IFC4. Compares before‑export schedules with after‑export IFC data (doors, columns, floors) using ifcopenshell. Computes record & parameter fidelity, generates charts, and outputs DFLR metrics. Ideal for BIM interoperability research and quality assurance.
